# Application Purpose

This platform automates guest access management for Airbnb and Booking.com properties using TTLock smart locks.

The system should:

* synchronize reservations from iCal feeds
* automatically generate temporary access PINs
* create TTLock access codes
* notify guests automatically
* revoke codes after checkout
* notify cleaners after guest departure

# Main Workflow

1. Host creates account
2. Host creates organization/workspace
3. Host adds property
4. Host connects TTLock smart lock
5. Host uploads Airbnb/Booking iCal URL
6. System periodically syncs reservations
7. When a new reservation appears:

   * generate temporary PIN
   * create TTLock access code
   * configure validity dates
   * email guest instructions
   * create cleaner task
8. After checkout:

   * automatically revoke PIN
   * notify cleaner
   * archive reservation activity
